== Description ==
This plugin is an account plugin. In this plugin, you can enjoy various features. In this plugin, you can use the login, registration, etc. forms of the plugin itself.

== This plugin has the following features: ==
* Custom login and register form
* Custom lost password and reset password form
* User register email notifications
* User support snd ticketing system
* User comments in frontend panel
* Send notifications by admin
* Edit the profile by the user
* Register form builder
* Panel menus manager
* Customizable order of all panel menus
* Custom menu icons
* Hide WP Admin
* Hide WP default login form
* Hide WP default signup form
* Hide WP admin bar for users
* other feathers

== Installation ==
1. Install the plugin
2. Activate the plugin
3. Use shortcodes mentioned in the "Plugin Info"

== Frequently Asked Questions ==
= How can i display the panel page? =
By Using shortcodes that mentioned in the "Plugin Info", in plugin menu
= How can i set custom icon for panel menus? =
You can use dashicon font icons of wordpress and any font icon packages that provides font icon by css classes and has used by your theme.
